Commands

[parameter] = Required

{parameter} = Optional

Moderation

/ban [user] {reason}

Bans a user from the server.

/unban [user]

Unbans a user from the server

/kick [user] {reason}

Kicks a user from the server

/warn [user] [reason]

Warns a user in the server

/cases [user]

Displays a users cases (warnings) in the server

/deletecase [case id]

Delete a specfic case from a user's profile

Economy

/balance

Get your balance of SeedCoins in the server

/daily

Get a daily gift of coins

/beg

Beg Seeds for a few measly coins

/highlow

Play the higher lower game to win coins

/slots [bet > 10]

Play the slot machines

/rps [bet > 10] [move]

Play rock paper scissors against Seeds

/shop

See items that are avaliable for purchase

/buy [shop id]

Buy something from the shop

/fish

Cast your pole out and hope for a bite

Fun

/friend [message]

Talk to an AI friend because you have none

/tsh [prompt]

Get a two sentence horror story

Utility

/stats

Get some stats about Seeds

/rcolor

Get a random color with the hex code
